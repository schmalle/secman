# Add AWS Account ID Column to Products Overview

## Context

The Products Overview page (`/products`) shows a table of systems running a selected product, with columns: **System Name**, **IP Address**, **Domain**. The `Asset` entity already has a `cloudAccountId` field (populated during CrowdStrike vulnerability imports), but it is not surfaced in the Products view. Adding it as the last column helps users identify which AWS account each system belongs to.
